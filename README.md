# GitNews 챗봇 만들기

RAG & Langchain 기술을 사용하여 GitNews에서 사용하는 챗봇을 생성하는 프로젝트입니다.
테그 기반 타임라인과 요약을 제공하고, 관련된 QA를 진행할 수 있습니다.
일반적인 QA의 경우 키워드를 기반으로 대화 및 질의가 진행됩니다.

- 프로젝트 기한 : 2025.08.25 ~ 08.29
- 프로젝트 주제 : 테그 및 키워드 기반 QA, 타임라인 제공
